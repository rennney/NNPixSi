* Introduction

This package provide framework to train deep learning models to perform signal proccessing for LArTPCs with Pixel based anodes.

Currently Implemented model is Graph Attention Neural Net

* Install

~NNPixSi~ installs in the "usual Python way".  Use of ~pip~ and a Python
virtual environment is recommended.

#+begin_example
python3 -m venv venv
source venv/bin/activiate
git clone https://github.com/rennney/NNPixSi.git
cd NNPixSi
pip install -e .
#+end_example

* Usage

The ~nnpixsi~ command line user interface provides online help:

#+begin_src shell :exports both :results output code :wrap example
nnpixsi
#+end_src

#+RESULTS:
#+begin_example
Usage: nnpixsi [OPTIONS] COMMAND [ARGS]...

  NNPixSi command line interface

Options:
  -s, --store PATH     File for primary data storage (input/output)

Commands:
